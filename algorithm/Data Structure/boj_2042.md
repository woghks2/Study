## **풀이**

### **방향성 생각**

* 세그트리 기본

* 구간합 구하는데는 펜윅트리가 더 빠르다고 해서 펜윅트리로 풀이

---

### **전체코드**

```python
import sys
input = lambda : sys.stdin.readline().strip()

def update(x,diff):
    while x <= N:
        tree[x] += diff
        x += x&(-x)

def cumsum(x):
    result = 0
    while x:
        result += tree[x]
        x -= x &(-x)
    return result

N,M,K = map(int,input().split())
arr = [0] + [int(input()) for _ in range(N)]
tree = [0]*(N+1)

for i in range(1,N+1):
    update(i, arr[i])

for _ in range(M+K):
    a,b,c = map(int, input().split())
    if a==1:
        diff = c - arr[b]
        arr[b] = c
        update(b,diff)
    else:
        print(cumsum(c)-cumsum(b-1))
```
