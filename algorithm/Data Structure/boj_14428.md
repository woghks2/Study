## **풀이**

### **방향성 생각**

* 세그트리

### **전체코드**

```python
import sys
input = lambda : sys.stdin.readline().strip()
DEFAULT = (float('inf'),float('inf'))

N = int(input())
arr = list(map(int,input().split()))
segtree = [DEFAULT]*(4*N)

def build(x,s,e):
    if s==e:
        segtree[x] = (arr[s],s)
        return segtree[x]
    m = (s+e)//2
    lv = build(2*x+1,s,m)
    rv = build(2*x+2,m+1,e)
    segtree[x] = min(lv,rv)
    return segtree[x]
    
def update(x,s,e,idx,val):
    if s==e:
        arr[idx] = val
        segtree[x] = (val,idx)
        return segtree[x]
    m = (s+e)//2
    if idx<=m:
        lv = update(2*x+1,s,m,idx,val)
        rv = segtree[2*x+2]
    else:
        lv = segtree[2*x+1]
        rv = update(2*x+2,m+1,e,idx,val)
    segtree[x] = min(lv,rv)
    return segtree[x]

def query(x,s,e,l,r):
    if e<l or s>r:
        return DEFAULT
    if l<=s and e<=r:
        return segtree[x]
    m = (s+e)//2
    lv = query(2*x+1,s,m,l,r)
    rv = query(2*x+2,m+1,e,l,r)
    return min(lv,rv)

build(0,0,N-1)

for _ in range(int(input())):
    a,b,c = map(int,input().split())
    if a==1:
        update(0,0,N-1,b-1,c)
    elif a==2:
        print(query(0,0,N-1,b-1,c-1)[1]+1)
```

### **코멘트**

* 인덱스 업데이트 될 때 min으로 비교해주기.
