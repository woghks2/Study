## **풀이**

### **방향성 생각**

* 구간합 구하는데는 펜윅트리가 더 빠르다고 해서 펜윅트리로 풀이

---

### **전체코드**

```python
import sys
input = lambda : sys.stdin.readline().strip()

N,M = map(int,input().split())

arr = [0]*(N+1)
tree = [0]*(N+1)

def update(x,diff):
    i = x
    while i<=N:
        tree[i] += diff
        i += i&(-i)

def cumsum(x):
    result = 0
    i = x
    while i:
        result += tree[i]
        i -= i&(-i)
    return result

for _ in range(M):
    cmd,a,b = map(int,input().split())
    if cmd:
        diff = b - arr[a]
        arr[a] = b
        update(a,diff)
    else:
        print(cumsum(max(a,b)) - cumsum(min(a,b)-1))
```
